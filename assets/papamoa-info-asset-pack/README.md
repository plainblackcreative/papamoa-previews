# pāpāmoa.info logo asset pack

## Included
- Transparent combined logo PNGs in multiple widths
- Transparent wordmark-only PNGs
- Transparent koru-wave icon PNGs in common square sizes
- favicon.ico with multiple embedded sizes
- Common browser and mobile icon files
- site.webmanifest
- Dark and light preview images

## Recommended website files
- Header logo: logo/papamoa-info-logo-transparent-800w.png
- Favicon: favicon/favicon.ico
- Apple touch icon: favicon/apple-touch-icon.png
- Android / PWA icons: favicon/android-chrome-192x192.png and favicon/android-chrome-512x512.png

## Notes
These are raster PNG assets created from the approved transparent logo artwork. A true editable vector SVG is not included.
