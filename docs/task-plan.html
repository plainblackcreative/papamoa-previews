✅ 7. Pre-Launch QA Checklist

0/0 Run last

Run this sweep across every page in the repo before the client demo. Aim for a single focused QA session — not piecemeal fixes.

🎨
CSS & Design System
Figtree + Playfair Display loading. All colour variables correct (--navy, --blue, --lime, --sand, --dune, --coral, --gold). No broken layouts at 375px, 768px, 1280px.
🔗
Internal Links
No /slug links — all should be /articles/slug, /listings/slug etc. No dead "advertise", "claim", or "Home" links. Nav brand link goes to /.
🖼️
Images & Media
No broken image tags. Hero images load. Logos render at correct size. No placeholder image paths from template that weren't replaced.
🖉
Editorial Voice
Spot-check 10 pages. No "we/us/our" in listing copy. No "contact us" — all should be "call them / visit them". Macron on Pāpāmoa. No em dashes.
📊
Forms & APIs
Web3Forms: submit a test from each active form type. Check email receipt. Cloudflare Worker: live test via PPP search checker. Admin upload: AI summarise test.
📱
Mobile Rendering
Every page tested on iPhone SE (375px). Two-col layouts collapse to single col. Tables scroll horizontally. No horizontal overflow. Nav wraps cleanly.
🔍
SEO Basics
Every page has a unique title tag. Meta description present. No duplicate H1s. noindex only on admin/sales pages. Sitemap reflects current repo state.
🔴
Accuracy & Dates
Mauao closure status correct (fully closed, April 2026). Pāpāmoa Hills track status correct (Summerhills open, all others closed). Mayor name correct (Mahé Drysdale). AIMS Games dates (5–11 Sep 2026).
