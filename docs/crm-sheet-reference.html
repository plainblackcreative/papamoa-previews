Step-by-step instructions to create this Google Sheet from scratch. Complete in order.
Step 1 — Create the Sheet
  1. Go to sheets.google.com
  2. Click Blank spreadsheet
  3. Click "Untitled spreadsheet" at the top → rename to Papamoa.info — Master CRM
  4. Rename Sheet1 to LEADS
  5. Click + at the bottom to add sheets → rename to LISTINGS and RENEWALS
Step 2 — Add Column Headers
Quick tip: Row 1 of each tab = headers. Type each column name exactly as shown in the tabs above. Bold row 1: select it → Format → Bold. Freeze row 1: View → Freeze → 1 row. Make the header row background colour navy (#2B3F5C) with white text for each tab.
Step 3 — Add Data Validation (Dropdowns)
  1. Click the column letter to select the whole column
  2. Data → Data Validation → Add a Rule
  3. Criteria: Dropdown
  4. Enter values from the table above, one per line
  5. On invalid data: Show a warning
  6. Click Done
  7. Repeat for each dropdown column
Step 4 — Add Formulas
LISTINGS tab — Renewal Date (column S): Click cell S2, type =P2+365, press Enter. Then click S2, copy it (Ctrl+C), select S2:S1000, paste (Ctrl+V). This calculates renewal date for every row automatically.
RENEWALS tab — Days Until Renewal (column D): Click cell D2, type =C2-TODAY(), press Enter. Copy down to D1000. Then select D2:D1000 → Format → Conditional Formatting → add the three rules shown in the Renewals tab.
Step 5 — Connect to Make.com (Phase 2)
  1. In Make.com: Create scenario → Add Google Sheets module
  2. Connection → Add → Sign in with [email protected] Google account
  3. Allow Make.com access to Google Sheets
  4. Select spreadsheet: Papamoa.info — Master CRM
  5. Select sheet: LEADS
  6. Make.com can now read, write, and update rows automatically
Step 6 — Share with Make.com Service Account (if needed)

If Make.com requires a service account email for access, share the sheet with that email address: File → Share → Share with people → enter the service account email → set to Editor.
