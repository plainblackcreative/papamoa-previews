RENEWALS Tab

Shows all listings with upcoming renewals. At launch, populate manually from LISTINGS tab. In Phase 2, Make.com checks this daily and sends renewal reminder emails automatically via Brevo.

Conditional Formatting — Days Until Renewal

Urgent — Red =D2<7
Renewal in less than 7 days. Call immediately.
Due Soon — Amber =AND(D2>=7,D2<=30)
Renewal in 7-30 days. Send reminder email.
On Track — Green =D2>30
More than 30 days. No action needed.

Automated Health Checks (GitHub Actions, weekly)

  • Website reachability: HTTP HEAD to each listing URL. Flags 404, 500, DNS failure, SSL expiry within 30 days.
  • Email deliverability: MX record lookup + SMTP check on listing email domains.
  • Phone format: Validates NZ landline (0X XXX XXXX) and mobile (02X XXX XXXX) patterns.
  • Google Business Profile hours: Compares listing hours vs GBP (where available). Flags mismatches.
  • Social profile links: Checks Facebook and Instagram URLs resolve to active pages.

Setup Instructions

Step-by-step instructions to create this Google Sheet from scratch. Complete in order.

Step 1 — Create the Sheet

  1. Go to sheets.google.com
  2. Click Blank spreadsheet
  3. Rename to Papamoa.info — Master CRM
  4. Rename Sheet1 to LEADS
  5. Click + at the bottom to add sheets: LISTINGS, RENEWALS, HEALTH

Step 2 — Add Column Headers

Row 1 of each tab = headers. Type each column name exactly as shown in the tabs above. Bold row 1: select it → Format → Bold. Freeze row 1: View → Freeze → 1 row. Make the header row background colour navy (#134B58) with white text for each tab.

Step 3 — Add Data Validation (Dropdowns)

  1. Click the column letter to select the whole column
  2. Data → Data Validation → Add a Rule
  3. Criteria: Dropdown
  4. Enter values from the table above, one per line
  5. On invalid data: Show a warning
  6. Click Done
  7. Repeat for each dropdown column across all tabs

Step 4 — Add Formulas

LISTINGS tab — Renewal Date (column U):

=R2+365 // Auto-calculates renewal date from payment date

Click cell U2, type the formula, press Enter. Copy down to U1000.

RENEWALS tab — Days Until Renewal (column D):

=C2-TODAY() // Shows days remaining, apply conditional formatting

Copy down to D1000. Then select D2:D1000 → Format → Conditional Formatting → add the three colour rules shown in the Renewals tab.

HEALTH tab — Next Verification Due (column D):

=C2+182 // 6 months from last verified date

HEALTH tab — Days Until Due (column E):

=D2-TODAY() // Same conditional formatting as Renewals

Step 5 — Connect to Make.com (Phase 2)

  1. In Make.com: Create scenario → Add Google Sheets module
  2. Connection → Add → Sign in with jkbrownnz@gmail.com Google account
  3. Allow Make.com access to Google Sheets
  4. Select spreadsheet: Papamoa.info — Master CRM
  5. Select sheet: LEADS
  6. Make.com can now read, write, and update rows automatically

Step 6 — Share with Make.com Service Account (if needed)

If Make.com requires a service account email for access, share the sheet with that email address: File → Share → Share with people → enter the service account email → set to Editor.
