Help & workflow

How this admin page works and what happens after you submit.

📰 News workflow

  • Paste a URL from any public news source
  • Click Summarise with Claude — the article is fetched and passed to Claude via the Cloudflare proxy
  • Claude writes a 2–3 sentence summary in the Papamoa.info editorial voice (third-person, factual)
  • Edit the headline and summary as needed
  • Click Submit for publishing — sends to jkbrownnz@gmail.com via Web3Forms
  • Manually copy the summary into news-events.html, or wait for the GitHub Actions workflow (Phase 2)

📅 Event workflow

  • Fill in the event details and click Submit event
  • Sends to jkbrownnz@gmail.com via Web3Forms
  • Review in email, then manually add to the news-events.html upcoming events section
  • Phase 2: GitHub Actions will automate this — submissions will write directly to a JSON file and the page rebuilds
