Bronze Listing Queue
Moderation queue for free Bronze listings. Reject marks the row rejected. Nothing goes live without approval. Requires the admin token + the Bronze Worker deployed (Project_Master §8.2 -- Bronze-as-card model locked 2026-06-04).
